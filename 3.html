평균값 정리

함수 f(x)가 닫힌구간 [a, b]에서 연속이고 열린구간 (a, b)에서 미분가능하면
인 c가 a와 b 사이에 적어도 하나 존재한다.

증명 sketch

  1. 롤의 정리와 달라진 점은 f(a)=f(b) 조건이 없다는 것이다.
  2. 그래서 우리는 적당한 함수 g(x)를 찾아 h(a)=h(b)=0를 만족하는 새로운 함수 h(x)=f(x)-g(x)를 정의할 것이다.
    새롭게 정의한 함수 h(x)는 롤의 정리의 조건을 모두 만족하게 되어, 롤의 정리를 적용할 수 있다.
  3. 그렇다면 g(x)를 어떻게 찾을 수 있을까? 우리는 f(x)에서 적당한 함수를 빼서, 즉, f(a)-g(a)=f(b)-g(b)인 함수 g(x)를 찾고싶다.
    수학자들이 찾은 g(x)는 (a, f(a)), (b, f(b))를 지나는 일차함수이다.
  4. 이렇게 g(x)를 정의했을 때 f(a)-g(a)=f(b)-g(b)=0을 만족하므로 함수 h(x)에서 h(a)=h(b)=0,
    즉 롤의 정리의 모든 조건을 만족하게 된다.
****이것을 기하적 관점으로 살펴보면 직선 g(x)를 x축으로 만드는 역할을 한다.****

증명

  1. 곡선 y=f(x)위의 두 점 P(a,f(a)), Q(b,f(b))를 지나는 직선의 방정식을 g(x)라고 하면
    이다.
  2. 이 때 h(x)=f(x)-g(x)라고 하면 함수 h(x)는 닫힌구간 [a, b]에서 연속이고
    열린구간 (a, b)에서 미분가능하며 h(a)=h(b)=0이다.
  3. 따라서 롤의 정리를 이용하면 h'(c)=0인 c가 a와 b 사이에 적어도 하나 존재한다.
    즉, 인 c가 a와 b 사이에 적어도 하나 존재한다.
